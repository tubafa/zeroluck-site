# О проекте

ZeroLuck (ZLUCK) — одностраничный лендинг маркетингового агентства.
Строим маркетинговые и коммерческие системы для бизнеса. Не агентство — системный интегратор.

- Рынки: Узбекистан, Азербайджан, СНГ
- Аудитория: владельцы бизнеса и CEO средних/крупных компаний
- Язык сайта: русский
- Основатель: Боходир Тураев

# Стек технологий

- Next.js 14 (App Router) + TypeScript
- Tailwind CSS
- Framer Motion + Intersection Observer (анимации)

# Дизайн-система

Цвета: акцент #001FFF, фон #000000, текст #FFFFFF / #B0B0B0
Шрифт: Classic Console Neue (моноширинный). Крупнее обычного — консольный шрифт нечитаем мелким.

Размеры текста (НЕ уменьшать):
- Десктоп 1024px+: body 20px/2.0, номера шагов 64px, заголовки шагов 28px
- Планшет 769-1023px: body 17px/1.8, номера шагов 52px
- Мобилка до 768px: body 15px/1.7, номера шагов 40px, заголовки шагов 22px

Визуал: грид-паттерн 40×40px (#001FFF, 8% opacity), scanline-оверлей, мигающие курсоры у заголовков, угловые скобки на карточках, частицы в hero, пульсация CTA каждые 0.8с.
Логотипы компаний: `filter: drop-shadow(0 0 8px rgba(0,31,255,0.6)) drop-shadow(0 0 20px rgba(0,31,255,0.3))`

# Правила

- Одна сессия Claude Code = одна задача
- НЕ уменьшать размеры шрифтов
- НЕ убирать грид-паттерн с фона — новые элементы через ::before/::after поверх
- ВСЕГДА проверять десктоп И мобилку
- Все тексты на русском
- Анимации быстрые и точные, не медленные
